# CursoApp

Objetivo: Um site para facilitar a parte de marketing do ULTRA CURSOS para gerar pdf e confirmar matrícula antes de ir para o sistema.

Como usar: Para usar, é necessario acessar o link "https://sitedaescola.netlify.app/" e preencher os campos. 

Como acessar o painel admin: Digitar a senha "admin123", se quiser mudar, ultilize o javascript do código, está localizado na parte "SENHAADMIN".
